# Online Coding Judge System

A scalable system similar to LeetCode and HackerRank.

## Features
- Multi-language code execution
- Secure sandbox environment
- Leaderboard system
- Scalable microservices architecture

## Tech Stack
- Backend: Node.js / Python (Flask)
- Database: PostgreSQL
- Cache: Redis
- Queue: Kafka
- Containers: Docker

## Architecture
Refer to high-level-design.md

## How to Run
1. Start database
2. Start backend server
3. Run worker nodes
